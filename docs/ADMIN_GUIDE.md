# Руководство по админке НА УГЛЕ

> Для контент-менеджеров и администраторов ресторана.
> Админка: https://landing.swipeandev.ru/admin

---

## Вход

- URL: `/admin/login`
- Логин: `admin@na-ugle.ru`
- Пароль: выдаётся при настройке (сменить после первого входа в `/admin/profile`)

---

## Дашборд (`/admin/main`)

Главный экран после входа.

**Что видно:**
- Кнопка **🟢 РЕСТОРАН ОТКРЫТ / 🔴 РЕСТОРАН ЗАКРЫТ** — один клик, управляет верхним баннером на сайте
- **Карточки быстрого доступа** — ссылки на все ключевые разделы
- **Метрики** — количество блюд, хитов, фото, заявок
- **Текст баннера** — можно быстро поменять прямо здесь
- **Последние заявки** — 5 свежих (если есть)

---

## Hero-слайдер (`/admin/hero-slides`)

Управляет каруселью на главной странице.

**Список:** таблица с превью фото, заголовком, слоганом, подзаголовком, статусом.

**Форма редактирования слайда:**

| Поле | Что делает |
|---|---|
| **Фотография** | Cropper 1920x1080. Кликните по миниатюре → выберите файл → кадрируйте. |
| **Заголовок (метка)** | Маленькая uppercase-метка: «01 / 03 · {Заголовок}». Пример: «Огонь и дым». |
| **Главный слоган** | Огромный текст поверх фото. Многострочный. **У каждого слайда свой.** |
| **Подзаголовок** | Описание под слоганом. Пример: «Живой огонь, хоспер и уголь». |
| **Текст кнопки** | CTA на слайде: «Смотреть меню», «Как нас найти» и пр. |
| **Ссылка кнопки** | `#menu`, `#contacts`, `/menu` — куда ведёт CTA. |
| **Порядок** | Число — определяет очерёдность в ротации. |
| **Активен** | Скрыть слайд из ротации без удаления. |

## Меню

### Категории (`/admin/menu/categories`)

10 категорий: На хлеб, Закуски, Салаты, Супы, Горячее, Колбасы, Гарниры, Соусы, Десерты, Напитки.

| Поле | Что делает |
|---|---|
| Название | Отображается на сайте. |
| Slug | Автогенерируется из названия. Используется в URL-якорях. |
| Иконка | Эмодзи (🥩, 🍰). Видна в навигации /menu и футере. |
| Порядок | Число — определяет очерёдность на сайте. |
| Активна | Скрыть всю категорию с сайта. |

**Важно:** удаление категории **удаляет все её блюда** (каскад).

### Блюда (`/admin/menu/items`)

78 позиций. Фильтры в списке: по категории, доступности, меткам (хит/шеф/без фото/без цены).

| Поле | Что делает |
|---|---|
| Категория | К какой категории относится блюдо. |
| Фотография | Загрузка через Picture. Видна на сайте в карточке. |
| Название | Отображается везде. |
| Описание | 1–2 предложения. Видно на /menu и в рекомендациях шефа. |
| Цена | В рублях. Если 0 → на сайте показывается «уточняйте». |
| **В наличии** | ⛔/✅ — моментально скрывает блюдо с сайта. |
| **Хит** | ⭐ — блюдо попадает в секцию «Хиты с хоспера» на главной + чип «★ Хит» на карточке. |
| **Рекомендует шеф** | 👨‍🍳 — попадает в «Рекомендации от шефа» + чип. |
| Комментарий шефа | Цитата от Андрея — видна только в блоке рекомендаций. |
| Порядок | Сортировка внутри категории. |

## Галерея (`/admin/gallery`)

Три секции фото:
- **Атмосфера** → горизонтальная галерея на главной
- **Блюда** → запас на будущее
- **Команда** → карусель рядом с блоком шефа

| Поле | Что делает |
|---|---|
| Фотография | Picture/Cropper. Загрузка файлом. |
| Секция | Выбор: Атмосфера / Блюда / Команда. |
| Alt-текст | SEO + доступность. Описание для поисковиков. |
| Порядок | Число. |
| Активна | Скрыть без удаления. |

## Шеф-повар (`/admin/chef`)

Одна запись (singleton). Данные для блока «Шеф» на главной.

| Поле | Что делает |
|---|---|
| Имя | «Андрей Воробьёв» |
| Должность | «Шеф-повар НА УГЛЕ и La Volte» |
| Фото | Портрет шефа. |
| Биография | Rich-text (Quill editor). |
| Цитата | Рукописным стилем на сайте. |
| Факты | Формат «Ключ: значение» по строкам (стаж, стажировки и пр.). |
| Заметка La Volte | Блок внизу о крем-брюле. |

## Настройки сайта (`/admin/settings`)

6 вкладок. **Кнопка «Сохранить всё»** сохраняет ВСЕ вкладки разом.

### Вкладка «Основное»
- Ресторан открыт/закрыт (баннер)
- Текст баннера
- Адрес, телефон, часы работы, сезон

### Вкладка «Манифест»
- Заголовок блока «Честная еда…» (многострочный)
- Основной текст
- Реплика Дурняши

### Вкладка «Заголовки секций»
- Галерея, Хиты с хоспера, Рекомендации шефа, Контакты — редактируемые заголовки

### Вкладка «Контакты и карты»
- Как добраться, ссылки на Яндекс.Карты/2GIS, embed-код карты, Instagram, Telegram

### Вкладка «PDF меню»
- **Загрузка файлом** (не URL) — выберите PDF, нажмите «Загрузить»
- Основное меню, Барная карта, Винная карта

### Вкладка «Уведомления»
- Email, Telegram Bot Token, Chat ID

## Что где менять (quick reference)

| На сайте | Где в админке |
|---|---|
| Баннер «Сегодня открыты» | Дашборд или Настройки → Основное |
| Hero-фото и слоган | Hero-слайдер |
| Текст «Честная еда…» | Настройки → Манифест |
| Фото атмосферы | Галерея (секция: Атмосфера) |
| Блюда меню + цены | Блюда меню |
| Хиты на главной | Блюда → тоггл «Хит» |
| Рекомендации шефа | Блюда → тоггл «Рекомендует шеф» + комментарий |
| Биография шефа | Шеф-повар |
| Телефон / адрес / часы | Настройки → Основное |
| PDF-файлы | Настройки → PDF меню |
| Заголовки секций | Настройки → Заголовки секций |
